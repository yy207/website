制造业ERP系统

客户: 某制造企业 时间: 2023年3月 标签: ERP, 制造, 管理系统

项目概述

该项目是为一家中型制造企业开发的ERP系统，旨在整合企业各部门业务流程，提高运营效率。系统包括生产计划、物料需求计划、采购管理、销售管理、库存管理和财务管理六大核心模块。

技术栈

  • 后端: Spring Boot 2.7, Spring Security, MyBatis-Plus
  • 前端: Vue 3, Element Plus, ECharts
  • 数据库: MySQL 8.0, Redis 6
  • 部署: Docker, Nginx, Jenkins

项目亮点

  • 实现了MRP(物料需求计划)算法，自动计算原材料需求
  • 开发了可视化生产排程看板，直观展示生产进度
  • 集成了条码扫描功能，提高库存盘点效率
  • 实现了多维度数据分析报表，支持决策制定
